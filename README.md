## 🎯 Quick Start

1. **Choose your project folder** (vedaaxis, vihaanbytes, etc.)
2. **Add your research** to the `research/` subfolder
3. **Generate content** and save to the `content/` subfolder
4. **Follow** the NotebookLM sync guide in `shared-tools/`

## 🔄 Workflow: NotebookLM → GitHub

1. **Research in NotebookLM** - Create your research, chat, and organize concepts
2. **Export from NotebookLM** - Copy research summaries and notes
3. **Organize in GitHub** - Paste into appropriate project folder
4. **Version Control** - Commit with clear messages
5. **Track Progress** - Use GitHub Issues for project management
